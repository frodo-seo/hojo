Hojo는 서버 없는 오픈소스 개인 금융 앱입니다.

스크린샷 한 장으로 지출·수입·자산을 기록하고, 카드·페이 결제 알림을 자동으로 읽어 장부에 쌓습니다. 모든 거래 내역과 설정은 기기 내부 IndexedDB에만 저장됩니다. 클라우드 동기화도, 계정 가입도, 중앙 서버도 없습니다. 운영자를 포함해 어느 누구도 사용자의 장부에 접근할 수 없습니다.

AI 기능(OCR 분류·파싱, 월간·연간 리포트)은 사용자가 직접 발급한 API 키로 동작하는 BYOK(Bring Your Own Key) 구조입니다. 앱 제공자의 인프라나 크레딧에 의존하지 않습니다.

<b>주요 기능</b>
• 스크린샷 단일 진입점 — 영수증·결제 알림·급여명세·청구서·증권·거래소 화면 자동 인식
• 결제 알림 자동 파싱 — 카드·페이·은행 푸시 알림을 Haiku가 읽어 금액·가맹점·카테고리 추출
• 5종 자동 분류 — 지출·수입·고정 지출·고정 수입·자산 매매
• 자산 포트폴리오 — 주식/ETF·암호화폐·원자재(금/은/플래티넘) 실시간 시세
• 기준통화 환산 순자산 합계 + 파이차트
• 월별 예산 설정 및 사용 현황
• 월간·연간 AI 리포트
• 한국어 / English UI + AI 리포트 언어 연동
• 저녁 9시 기록 리마인더
• CSV 내보내기

<b>필요한 API 키</b>
• Anthropic — 분류·파싱·리포트 (console.anthropic.com)
• Datalab — OCR (datalab.to)

API 사용료는 사용자 본인이 부담합니다. 일반적인 개인 사용 범위에서는 매우 저렴합니다.
