Hojo는 서버 없는 오픈소스 가계부입니다.

모든 거래 내역과 설정은 기기 내부 IndexedDB에만 저장됩니다. 클라우드 동기화도, 계정 가입도, 중앙 서버도 없습니다. 운영자를 포함해 어느 누구도 사용자의 장부에 접근할 수 없습니다.

AI 기능(영수증 OCR, 월간·연간 리포트)은 사용자가 직접 발급한 API 키로 동작하는 BYOK(Bring Your Own Key) 구조입니다. 앱 제공자의 인프라나 크레딧에 의존하지 않습니다.

<b>주요 기능</b>
• 지출·수입 기록 (카테고리·메모·날짜)
• 고정 수입/지출 관리
• 월별 예산 설정 및 사용 현황
• 영수증 카메라 OCR 자동 파싱
• 월간·연간 AI 리포트
• 저녁 9시 기록 리마인더
• CSV 내보내기

<b>필요한 API 키</b>
• Anthropic — AI 리포트·영수증 파싱 (console.anthropic.com)
• Datalab — 영수증 OCR (datalab.to)

API 사용료는 사용자 본인이 부담합니다. 일반적인 개인 사용 범위에서는 매우 저렴합니다.

<b>오픈소스</b>
